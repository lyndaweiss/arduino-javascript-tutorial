Software Installation

  1. Node.js

    Download Node from the Node.js website. The page will link to the download for your OS, but if you need a different version, you can click on "Downloads" in the top navigation menu.

    Installing Node on Windows is straightforward - download then run the installer. Installing Node on Linux may vary with distribution - there's more information available on the Node.js Download page.

  2. Node.js Modules

    We need to download and install two Node.js modules. We'll use the Node Package Manager (npm) to do this, so make sure you have Node.js downloaded and installed first. Also, this installation is done from a command line, so start up a Command Prompt (Windows) or shell (Linux) and then type the following commands.

    • mkdir blink - Create a directory for this project.
    • cd blink - Change to your project directory
    • npm install serialport - Download and install the SerialPort package, used by Node to communicate with the USB port.
    • npm install socket.io - Download and install socket.io package, used by Node to communicate with the client.

    When the above is finished, your project directory will have a subdirectory called "node_modules" that contains what you just installed along with other modules that SerialPort and socket.io depend on.

  3. Arduino IDE

    Download the version for your operating system from the Arduino website. To install, follow the installation instructions for your platform on the Arduino Getting Started page.
