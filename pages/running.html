Running the Example

To test the example, run the following steps. Make sure to perform them in the order shown, because the Arduino IDE and the Node server use the same USB port.

  1. Verify (compile) and Upload Arduino sketch - see Figure 1.
  2. Change to your directory for this project and start the server - see Figure 2.
  3. Open the client in a browser, using the URL (IP address and port number) given to the HTTP server (see the Server section).
  4. Enter a number of times to blink, a blink delay, and press the Blink Button
