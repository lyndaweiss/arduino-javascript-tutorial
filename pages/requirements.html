Requirements

It will help to have some prior knowledge of HTML, CSS, JavaScript, and writing Arduino sketches. We'll cover building the web page, writing the client and server scripts, and writing and downloading the Arduino sketch in the following sections. You can also download the example from GitHub

To build and run this example you will need the following:

Hardware

Arduino UNO

Breadboard

USB A-B Cable

5mm LED

220 Ω Resistor

Jumper Wires

Software

  1. Text or Code Editor, for writing HTML and JavaScript. I use and highly recommend Visual Studio Code (VSCode) - Download it here
  2. Node.js, JavaScript runtime environment that we will use to run our server.
  3. Arduino Integrated Development Environment (IDE), for writing and uploading the Arduino code.

We'll cover installing the software in the next section.
